# Item API

A simple Java backend application using Spring Boot that implements a RESTful API for managing a collection of items.

## **Description**

This project provides two main functionalities as per the sample task:

1. Add a new item
2. Get a single item by ID

The data is stored in-memory using an `ArrayList`. Input validation is applied for required fields.

## **Technologies Used**

- Java 17 (simple Java, no advanced features)
- Spring Boot 3.2.5
- Spring Web
- Jakarta Validation (for input validation)
- Maven
- In-memory ArrayList for data storage
