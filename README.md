# Item API

A simple Java backend application that implements a RESTful API for managing a collection of items (like ecommerce apps or movies). This project uses Spring Boot and an in-memory data store.

---

## Features

- Add a new item
- Get a single item by ID
- Input validation to ensure required fields are present

---

## Technology Stack

- Java 17
- Spring Boot 3.2.5
- Maven
- H2 Database (in-memory)
- Jakarta Validation
